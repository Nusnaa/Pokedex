# Pokedex

## How to Run

### Git Clone

```bash
git clone https://github.com/Nusnaa/Pokedex.git
```

### Install dependencies

You can use one of them `npm`, `yarn`, `pnpm`, `bun`, Example using `npm`:

```bash
npm install
```

### Run the development server

```bash
ng serve
```

### Go to URL

```bash
http://localhost:4200/Pokedex/
````
